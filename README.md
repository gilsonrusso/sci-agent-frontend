# SciAgent Frontend

This is the frontend for the SciAgent platform, built with React, TypeScript, and Vite.

## Prerequisites

- Node.js (v18+)
- npm or yarn

## Setup

1.  Navigate to the directory:

    ```bash
    cd sci-agent-frontend
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

## Development

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

## Features

- **Dashboard**: Manage projects.
- **Editor**: Real-time collaborative LaTeX/Markdown editor with auto-save.
- **Chat**: AI assistant integration.
- **Preview**: Live PDF preview via compilation service.

## Key Technologies

- React + Vite
- Material UI (MUI)
- CodeMirror 6
- Yjs (Real-time collaboration)
- TanStack Query
- Zustand (State Management)
